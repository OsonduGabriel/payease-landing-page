# PayEase — Fintech Landing Page

A fully responsive landing page for **PayEase**, a fictional Lagos-based fintech startup offering instant money transfers for small business owners.

Built as part of a frontend development project to practice React, responsive design, and real-world UI thinking.

🔗 **Live site:** https://payease-landing.netlify.app/

## Features

- Responsive design — mobile first, works on all screen sizes
- Hero section with CTA and trust indicators
- Features section highlighting key product benefits
- How It Works section with 3-step flow
- Testimonials from fictional Nigerian small business owners
- Sign up form with validation, toast notification, and success state
- Sticky navbar with hamburger menu on mobile
- Deployed on Netlify via GitHub

## Process

This project followed a deliberate build process:

1. **Research** — studied real Nigerian fintech sites (Paystack, Flutterwave, OPay, Kuda) before writing any code
2. **Copy first** — wrote headline, features, and testimonials before designing
3. **Component structure** — separated data from UI using a central `ProjectData.jsx` file
4. **Mobile first** — wrote base CSS for mobile, then expanded with media queries
5. **Deploy** — pushed to GitHub and connected to Netlify for automatic deployment
